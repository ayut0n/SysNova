## 📌 О проекте

**SysNova** — это ультралегкая программа на C++ для оптимизации дискового пространства и работы со сложными файлами в ОС Windows. Программа обладает кастомным графическим интерфейсом (без стандартных рамок Windows), поддерживает смену тем оформления и работает напрямую с Windows API для достижения максимальной эффективности.

## ✨ Ключевые возможности

### 🧹 Очистка от мусора
* **Точечная очистка:** Удаление файлов из системного `Temp`, пользовательского `%temp%`, папки `Downloads` и `Prefetch`.
* **Очистка кэша приложений:** Удаление мусора от браузеров и программ (`CrashDumps`, `INetCache`, `WebCache`).
* **Очистка корзины** в один клик.
* **Глубокая очистка диска:** Фоновый вызов встроенной утилиты Windows `cleanmgr /verylowdisk` для максимального освобождения места.
* Кнопка **«Очистить ВСЁ»** для комплексного применения всех функций разом.

### 💀 Принудительное удаление (Force Delete)
Инструмент для мгновенного уничтожения файлов и папок, которые «заняты другим процессом» или защищены системой:
* **Restart Manager API:** Динамически находит и принудительно завершает процессы, которые блокируют целевой файл.
* **Перехват прав:** Автоматически выполняет команды `takeown` и `icacls` для получения полных прав администратора на удаляемый объект (полезно при обходе защиты TrustedInstaller).
* Интеграция с нативным проводником Windows (`IFileOpenDialog`) для удобного выбора файлов и папок.

### 🎨 Современный интерфейс
* Написан на невероятно быстром фреймворке **Dear ImGui** (бэкенд GLFW + OpenGL3).
* Кастомное «безрамочное» (frameless) окно со скругленными углами и плавной анимацией загрузки.
* **Кастомизация:** Встроенная палитра для изменения акцентного цвета интерфейса (настройки сохраняются в `sysnova.ini`).
* **Локализация:** Поддержка русского и английского языков.

## 🛠️ Стек технологий

* **Язык:** C++17
* **Компилятор:** MinGW-w64 (MSYS2)
* **Сборка:** CMake
* **Графика и Окна:** [Dear ImGui](https://github.com/ocornut/imgui), [GLFW](https://github.com/glfw/glfw), OpenGL3
* **Системные API:** Windows API (Restart Manager, COM, Shell32, Advapi32, Psapi)

## 🚀 Как собрать проект (Среда разработки)

Проект настроен для сборки в **VS Code** с использованием компилятора **GCC (MSYS2/MinGW)**.

### 1. Подготовка окружения
1. Установите [MSYS2](https://www.msys2.org/).
2. Откройте терминал `MSYS2 MinGW x64` и установите необходимые пакеты (компилятор, CMake и GLFW):
   ```bash
   pacman -S mingw-w64-x86_64-toolchain
   pacman -S mingw-w64-x86_64-cmake
   pacman -S mingw-w64-x86_64-glfw
